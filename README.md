# 📊 Marketing Campaign Performance

## 📌 Project Overview

This project analyzes the performance of different marketing campaigns using **data analysis and A/B testing techniques**.
The goal is to understand which **marketing channels, campaign types, and customer segments** perform better in terms of **conversion rate and ROI**.

---

## 🎯 Objectives

* Analyze marketing campaign performance
* Compare marketing channels using **A/B testing**
* Identify the **best performing marketing channel**
* Study **customer segment behavior**

---

## 🛠 Technologies Used

* **Python**
* **Pandas** – data analysis
* **Matplotlib** – data visualization
* **Seaborn** – statistical visualization
* **SciPy** – statistical testing (T-test)

---

## 📂 Dataset

The dataset contains information about marketing campaigns such as:

* Campaign Type
* Channel Used
* Customer Segment
* Impressions
* Clicks
* Conversion Rate
* Engagement Score
* Acquisition Cost
* ROI
* Date

---


### 📊 New Metrics Created

* **CTR (Click Through Rate)** = Clicks / Impressions
* **CPC (Cost Per Click)** = Acquisition Cost / Clicks

---

## 📈 Business Insights

From the analysis we identify:

* **Best Performing Marketing Channel** – channel with highest average conversion rate
* **Highest ROI Channel** – marketing channel generating maximum return on investment
* **Best Customer Segment** – customer group with the highest conversion rate
